# libshgif

[Cj-bc/shgif](https://github.com/Cj-bc/shgif)用のライブラリです。
[Cj-bc/blib](https://github.com/Cj-bc/blib)を使用してインストールしてください。

# インストール方法

```bash
$ blib install Cj-bc/libshgif
```
